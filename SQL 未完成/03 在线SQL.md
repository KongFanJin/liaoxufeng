# 03 在线SQL

为了便于在线练习，我们提供了一个在线运行SQL的功能。实际上这是在浏览器页面运行的一个JavaScript编写的内存型SQL数据库[AlaSQL](http://alasql.org/)。不必运行MySQL等实际的数据库软件，即可在线编写并执行SQL语句。

可以在此测试执行最简单的SQL语句：

```sql
-- 以双减号开头的是注释
SELECT * FROM students;
```
